# ShadowMonitor

Lightweight SOC monitoring simulator built with Python.

ShadowMonitor is a terminal-based monitoring tool designed to simulate basic Security Operations Center (SOC) activities while practicing automation, system monitoring, and cybersecurity-related scripting.

The project continuously collects real-time system information and displays it through a lightweight command-line dashboard.

---

# Features

- Real-time CPU monitoring
- RAM usage tracking
- Active network connection analysis
- Automatic alert generation
- Live terminal dashboard
- Continuous monitoring loop

---

- Python 3
- psutil
- socket
- datetime
- os

# Installation

Clone the repository:

```bash
git clone https://github.com/pro/ShadowMonitor.git
```

Enter the project directory:

```bash
cd ShadowMonitor
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

# Usage

Run the script:

```bash
python main.py
```
